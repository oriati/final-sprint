# Twix
Final project at coding academy - a web page builder app

latest build ver http://twix-app.herokuapp.com/

# Server side (cd Rest-server/)
install dependencies
``
npm install
``

server with hot reload at localhost:3003
``npm start``


# Twix (cd Twix-app/)

install dependencies
``
npm install
``

run Twix
``
npm run dev
``
